# 🌟 Mundo das Palavras

Jogo educativo interativo desenvolvido para crianças do 2º ano do Ensino Fundamental, com foco na alfabetização e no reconhecimento dos fonemas **M, N e L**.

O jogo trabalha:
- Formação de palavras
- Consciência fonológica
- Associação entre letras e sons
- Leitura de palavras simples
- Coordenação visual e atenção

---

# 🎮 Como Funciona

A criança deve montar corretamente cada palavra utilizando as letras disponíveis na tela.

Ao acertar:
- Ganha pontos ⭐
- Recebe feedback visual animado
- Escuta uma narração positiva 🔊
- Avança automaticamente para a próxima fase

---
